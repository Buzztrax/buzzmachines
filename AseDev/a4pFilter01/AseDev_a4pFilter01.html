Description:
The [asedev a4pFilter01] machine can be used as a resonant 24dB Filter. The user can choose the cutoff and the amount of resonance he wants just by using the two parameters called "Cutoff" and "Resonance". This machine works like the [asedev a2pFilter01] machine except it is more aggressive in the result the user can recieve. If your interested in a less aggressive sound check out the [asedev a2pFilter01] machine. When using extreme settings you might get some heavy distortion.

Parameters:
Cutoff:            The parameter ranges from 1 to 22050. Where a value of 1 is a cutoff frequency of 1Hz and a value of 22050 is a cutoff of 22050Hz. The default cutoff is 5000Hz.
Resonance:     The parameter ranges from 1 to 1000. Where a value of 1 means no resonance and a value of 1000 let the machine get freaky.
